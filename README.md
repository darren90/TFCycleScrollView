# TFCycleScrollView

----

全新的轮播图控件，可循环轮播，可做广告位，，， 使用起来方便，易用。


	1.采用UICollectionView实现，代码简单，使用方便。
	2:图片加载采用的是主流的SDWebImage，无缝切换到项目中
